# FUTURE_ML_02
# Support Ticket Classification & Priority Tagging (Task 2)

## 📌 Project Overview
This project was developed as part of my Data Science Internship at **Future Interns**. The goal is to automate the processing of customer support tickets by using Natural Language Processing (NLP) to categorize issues and assign urgency levels.

### 🛠️ Key Features
- **Automated Classification:** Categorizes tickets into types (e.g., Billing, Technical Support, Software, Hardware).
- **Smart Priority Tagging:** Automatically assigns priority levels (Low, Medium, High, Critical) based on ticket content.
- **NLP Preprocessing:** Implemented a text-cleaning pipeline using NLTK to remove noise (stopwords, punctuation) and normalize text.
- **Machine Learning:** Utilized **TF-IDF Vectorization** and the **Random Forest Classifier** for multi-class prediction.
